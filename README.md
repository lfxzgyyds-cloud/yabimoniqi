亚比模拟器 · Yabi Simulator

> 一台像素风、Y2K 复古 Windows 风格的「自我状态 RPG 游戏机」。
> 你每天打开它，记录自己做了什么，屏幕里的那只小史莱姆就会随你的状态变色、变表情、写进战斗日志。
> **数据只存在你这台设备，不上传任何服务器。**

---

## 一、这是什么

亚比模拟器把一个有点抽象的东西——「我今天状态怎么样」——做成了一台游戏机：

- 屏幕里有一只像素史莱姆（亚比），它是你的状态化身。
- 你记录一个动作（喝了魔爪 /OD/ 加了班 / 哭了 / 早睡……），它的各项数值就会变化。
- 数值变了，史莱姆的颜色和表情也跟着变；你还能看到「今天都发生了什么」的战斗日志。

**核心哲学**：每天打开都是全新的一天。不考核、不排名、不制造焦虑——归零也只是一句「没什么大不了的，明天还能重新充」。

## 二、怎么开始

1. 用任意设备手机/电脑/平板自带浏览器打开 `亚比模拟器.html`（直接双击就行）。
2. **第一次打开**会让你设置一个「代号」（昵称），例如 `阿废 / pixel / 01`，留空会自动叫你「无名氏」。
   - 这个昵称会显示在角色名上，也会跟着你的存档一起备份。
3. 进入后你会看到一台「游戏机」：上方状态栏 + 一只史莱姆 + 一排状态条 + 底部导航。

> 💡 想用「本地存档」长期保留数据，建议通过本地地址打开（例如 `http://127.0.0.1:8080/原型.html`）。只要同一台设备、同一个浏览器，关掉再开数据都还在（见第四节）。

## 三、四个页面怎么玩

底部有四个按钮：**🏠 首页 / 🎮 记录 / 📜 时间线 / 📊 状态**。

### 🏠 首页（状态面板）
- 中间的史莱姆是你的状态化身，会随数值变色（见第五节）。
- 上方显示 `DAY`（第几天）和连续天数 `streak`。
- 一排状态条展示 10 项核心属性（见下文）。
- 点史莱姆下方的按钮可以快速进入「记录 / 时间线 / 状态」。

### 🎮 记录（核心玩法）
1. 点「🎮 记录」或首页的「▷ 记录动作」。
2. 选一个分类（🍔 饮食 / 🌙 睡眠 / 💢 情绪 / 🚿 身体 / 💬 社交 / 💼 学习工作 / 🛍️ 消费 / 📱 上网 / ⚠ 危险状态），也可以直接用顶部搜索框搜动作名（比如「加班」「喝魔爪」）。
3. 点一个动作卡片 —— 会弹出 `ACTION LOGGED` 反馈框，告诉你这次动作改了哪些数值。
4. 点 **「继续记录」** 会停在原位置继续选（不会跳回顶部）；点「回到首页」回主界面。
5. 每次记录都会**自动存进本机**，不用手动保存。

### 📜 时间线（今日战斗日志）
- 按时间顺序列出你今天记录的所有动作，以及每次的数值变化。
- 这是你「今天这一局」的回放。

### 📊 状态（状态详情）
- 列出 10 项核心属性 + 2 项隐藏属性（PRESSURE 压力值、HEARTBEAT 心跳）的当前数值。
- 给出当前状态的「诊断文案」和针对性的小建议（比如 HP 低就提示你去吃饭/休息）。

## 四、存档与备份（重要）

**日常不需要手动存档、也不需要每次导入。**
- 每次记录动作，系统都会把当前状态写进浏览器 `localStorage`。
- 下次打开会自动读回，关掉再开数据都还在。

**那「数字备份码」是干嘛的？** —— 它是给这些情况的「逃生舱」：
- 换设备 / 换浏览器
- 清了浏览器数据
- 想把存档发给朋友

操作方式：
1. 首页点 **「🔐 备份码 / 本地存档」** → 「⧉ 复制数字码」导出一段**纯数字**（你全部存档的本地编码，含昵称、天数、日志）。
2. **导入**：在**第一次打开（设置昵称）的界面**下方，有「已有数字备份码？直接导入」框，把数字码粘进去就能整体恢复（含昵称）。
   - 校验失败会提示「可能抄错几位」，不会静默出错。
3. 同一个弹窗里还有「清除本机存档」（二次确认），把数据从当前设备删掉。

> ⚠️ 备份码是「本地混淆 + 数字兑换」，设计目的是让你能用一串数字随身带走自己的存档，**不是高强度加密**，也绝不上云。

## 五、史莱姆的 6 种颜色

角色会根据你的数值实时变色（不会永远绿色）：

| 心情 | 颜色 | 触发条件 |
|------|------|----------|
| **稳定 ok** | 荧光绿 | 各项正常（默认状态） |
| **兴奋 hype** | 橙色 | 兴奋度 HYPE 偏高 |
| **过载 chaos** | 紫红 | 混乱值 CHAOS 偏高 |
| **低精神 sad** | 深蓝 | 精神值 SAN 偏低 |
| **低血量 dead** | 暗红粉 | 血量 HP 偏低 |
| **归零 zero** | 白色 + 头顶金色光环 😇 | 任意属性 = 0（戏谑路线，不制造焦虑） |

归零时不会报警，而是用一句无所谓口吻的文案，比如「没什么大不了的，明天还能重新充」。

## 六、核心属性速查

| 属性 | 含义 | 怎么涨 |
|------|------|--------|
| HP 血量 | 身体电量 | 吃饭、早睡、运动 |
| SAN 精神值 | 精神状态 | 散步、洗澡、早睡、被夸 |
| HYPE 兴奋度 | 兴奋 / 上头 | 喝魔爪、赢游戏、冲动消费 |
| CHAOS 混乱值 | 混乱 / 失控 | 熬夜、发疯、刷短视频 |
| AWAKE 清醒度 | 清醒程度 | 喝咖啡、午睡 |
| HUNGER 饥饿值 | 越饿越高 | 没吃饭会涨；吃东西降 |
| CLEAN 洁净度 | 干净程度 | 洗澡、刷牙 |
| SOCIAL 社交能量 | 社交电量 | 聊天、独处充电；被迫社交掉 |
| MONEY 金钱值 | 钱包 | 发工资、交付项目；消费掉 |
| CONTROL 自控力 | 自律 | 完成任务、省钱、断网 |

所有数值 **0–100 钳制**。还有两项看不见的属性：**PRESSURE 压力值**、**HEARTBEAT 心跳**（例如「哭了」会降压力，「焦虑」会加速心跳）。

## 八、每日机制（DAY 跟着真实时间走）

- 第一次打开记为 `DAY 1`，之后**跟着你手机/电脑的真实日期走**：过了系统 0:00，`DAY` 自动 +1。
- 跨天且昨天也打开 → 连续天数 `streak` +1；断了就归 1。
- 每天打开时，状态会从「起床基准」重新算起（像每天都是新的一局），契合「每天都是全新的一天」。
- 跨 0:00 后首次打开会有一段开机过场（DAY N + 开屏语 + 连续存活天数）。

## 九、常见问题

**Q：我要不要每次都点「存档」？**
A：不用。每次记录都自动存本机，关掉再开还在。

**Q：我要不要每次都「导入」？**
A：不用。导入只在换设备 / 清数据 / 收朋友存档时用。

**Q：清了浏览器缓存数据会没吗？**
A：会。所以重要存档请用「🔐 备份码」导出那段数字码留存。

**Q：这是游戏还是真记录工具？**
A：都是。它用游戏外壳承载真实的自我观察，但不考核、不排名，主打一个「陪你活着」。
